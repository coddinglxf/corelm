# PrimeLM #

This is the best neural network language model ever!

## 1. Features ##

* Theano stuff (GPU, ...)
* Monolingual and bilingual LM
* SMT support (Moses integration and re-scoring)

## 2. Installation ##

Just download the project.

## 3. Getting Help ##
